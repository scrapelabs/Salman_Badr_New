What each mode does

Production
  • Real-time test button submits jobs normally.

  • This is the default for every API.
Maintenance
  • Real-time test button is blocked with a clear message.
  • Public API returns HTTP 503 with {"errors":[…], "maintenance":true, "message":…}.
  • An amber strip appears on every other tab so the state can't be missed.
